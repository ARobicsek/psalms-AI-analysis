2. ✓ How to Run the Pipeline on a New Chapter
The pipeline script is: run_enhanced_pipeline.py Basic Usage:
python scripts/run_enhanced_pipeline.py PSALM_NUMBER
Examples:

# Generate complete commentary for Psalm XX
python scripts/run_enhanced_pipeline.py XX

#just all word docs
python scripts/run_enhanced_pipeline.py XX --skip-macro --skip-micro --skip-writer --skip-copy-editor
OR
test_fix_XX.py

python scripts/concordance_tool.py
